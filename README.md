# Що робить компонент:

Генерує пов’язану пару діаграм (Use Case + Sequence) для ключової функції Create Reservation (за замовчуванням),

Дозволяє обрати інші функції з випадаючого списку,

Експортує обидві діаграми в один PNG-файл.

# Як використовувати:

Відкрий код у своєму React-проєкті (файл створено у редакторі).

Імпортуй і використай компонент у сторінці (наприклад, <UmlDrawer />).

Переконайся, що в проєкті доступна Tailwind або заміни класи на свої стилі.
